Download instructions:

For windows users: **once downloaded make sure to extract all files before continuing. 
Once extracted go into the folder named ‘Project-KNF(ver. 3 Windows)’ and look for the 
exe file name ‘Project KNF.exe’. From there the game is playable.**

For Mac users: ** once the exe file is downloaded it should appear in your downloads folder. 
Click the exe named ‘Project-KNF (ver. 3 Mac)’ and it should open by itself. You are then free to play**

All code for the game is under the asset file in the script sub file
